# Amativeness

A beautiful Wordpress theme, based on [Amativeness 3.0](http://azfashao.com/amativeness3-0/).

## Development

```
$ npm install
$ npm install -g gulp
$ gulp release
```

## 相关说明

本主题不适合新手使用，使用前请确保具有前端、PHP的基础。

顶部头像：后台->外观->顶部->上传

依赖的插件（必装）：Post Ratings、WP-PostViews
